## üöÄ Getting Started

### Installation

```bash
npm install -g claude-symphony@1.0.0
# or
pnpm add -g claude-symphony@1.0.0
```

### Initialize a New Project

```bash
claude-symphony init my-project
cd my-project
```

### Use Sub-Agents

```bash
# Validate current stage outputs
/validate

# Generate HANDOFF for stage transition
/handoff

# Synthesize parallel AI outputs
/synthesize

# Review architecture for issues
/arch-review
```

### Auto-Triggered Agents

Agents automatically run during workflow:
- **Stage transition** (`/next`): Auto-triggers handoff + synthesis + validation
- **5 tasks completed**: Auto-triggers checkpoint-manager-agent
- **Stage completion**: Auto-triggers validation-agent
